Class AbstractSensorControl<SensorType extends ISensorModule<?>>

  • java.lang.Object
    • org.sensorhub.impl.sensor.AbstractSensorControl<SensorType>

    public abstract class AbstractSensorControl<SensorType extends ISensorModule<?>>
extends Object
implements ISensorControlInterface

    Default implementation of common sensor control interface API methods. By default, async exec, scheduling and status history are reported as unsupported.
