Interface IObsStorage

  • All Superinterfaces:
    IBasicStorage
    All Known Subinterfaces:
    IObsStorageModule<ConfigType>
    All Known Implementing Classes:
    GenericStreamStorage

    public interface IObsStorage
extends IBasicStorage

    Adds support for storing/retrieving features of interests.
    ObsKey and IObsFilter can be used in all record retrieval methods and concrete implementations must handle them properly (e.g. they must handle all obs filter criteria when they are specified)

    Since:
    April 25, 2015
